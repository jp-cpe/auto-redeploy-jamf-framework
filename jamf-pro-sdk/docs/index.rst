Jamf Pro SDK for Python
=======================

A Python library for developing against the Jamf Pro APIs.

.. note::

   This project is under active development.

If you are writing scripts or building applications that interact with the Jamf Pro Classic or Pro
APIs proceed to the **User Guide** to get started. If you are building webhook-based integrations
for real-time operations proceed to the **Webhooks** section to learn about the tools in this SDK
to assist you.
